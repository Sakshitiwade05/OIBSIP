# Task 3 - To-Do WebApp

###  Description
A simple To-Do WebApp created using HTML, CSS, and JavaScript.  
Users can add daily tasks, mark them as completed, and delete them easily.

###  Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Auto display date & time when a task is added
